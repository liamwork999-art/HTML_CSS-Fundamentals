# 🌐 HTML & CSS Fundamentals

A personal learning repository dedicated to building a strong foundation in front-end web development. This workspace contains practice files, exercises, and small projects focused on HTML structure, CSS styling, layout systems, responsiveness, and visual design.

## ✨ What this repository includes

This journey covers:

- semantic HTML and page structure
- forms and user input elements
- common HTML content elements
- CSS fundamentals and styling techniques
- the box model and positioning
- flexbox for modern layouts
- responsive design with media queries
- advanced selectors and CSS features
- CSS grid for complex layouts
- transitions, animations, and light interactive styling

## 📚 Learning path

The repository is organized into progressive lessons:

1. Essential HTML
2. Forms and Inputs
3. More HTML Elements
4. CSS Basics
5. Box Model and Positioning
6. Flexbox
7. Responsive Media Queries
8. Various CSS Features
9. Advanced Selectors
10. CSS Grid
11. Transitions, Animations, and JavaScript Basics

## 🧪 How to use this repository

1. Open the folder in VS Code.
2. Browse the lesson folders in order.
3. Open any `.html` file and preview it in the browser.
4. Use Live Server for a smoother development experience.

## 🎯 Goals

The purpose of this repository is to help build confidence in:

- creating clean and readable HTML
- styling pages with CSS effectively
- understanding layout and spacing
- making designs responsive and polished
- preparing for JavaScript and frontend frameworks

## 📁 Repository structure

- `00_practice` — small exercises and experimentation
- `01-essential-html` — core HTML concepts
- `02-forms-and-inputs` — forms and input styling
- `03-more-html-elements` — additional semantic and structural elements
- `04-css-basics` — foundation styling with CSS
- `05-box-model-and-positioning` — layout and spacing fundamentals
- `06-flexbox` — modern flexible layouts
- `07-responsive-media-queries` — responsive design techniques
- `08-various-css-features` — additional styling tools
- `09-advanced-selectors` — more powerful CSS targeting
- `10-css-grid` — grid-based layouts
- `11-transitions-animations-js` — motion and interaction basics

## 💡 Why this matters

HTML and CSS are the foundation of every website. Mastering them makes it easier to build attractive, accessible, and responsive web pages before moving into JavaScript and modern frameworks.
